# QR Code Generator

A QR code generator: turn any URL or text into a scannable QR code and download it as PNG, all in the browser. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="qr-code.js"></script>
<qr-code-generator></qr-code-generator>
```

## What it does

The tool encodes your text to the QR specification with Reed-Solomon error correction (level M), so the code scans reliably even with minor print wear. Everything runs in the browser, so whatever you encode never leaves your device.

## When you'd use it

Linking print to web — posters, packaging, business cards, table tents.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
